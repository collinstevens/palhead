# Status effects

In-game **Survival Guide → Tips → Status Effects** text, captured from screenshots.

Shipped on **`/guides/status-effects/`** (normalized into `data/normalized/status-effects.json` at build time).

| File | Role |
|------|------|
| `status_effects.json` | Exact Survival Guide wording + provenance |
| `evidence/` | Permanent screenshot archive (never delete/overwrite) |

**Source of truth:** the Survival Guide tip, not wiki sites. If the game text changes, add new evidence and update the JSON — keep prior shots.

Path to open in-game: **Options → Survival Guide → Tips → Status Effects**.
