## How to use

1. Prefer section **1** and **2** when picking Palpedia screenshots.
2. After a screenshot, archive it under `corrections/evidence/` (never delete).
3. Record truth in `corrections/corrections.json` and mark the site wrong in `known-inaccuracies.json`.
4. Update `checklist.json` for that pal.
5. Rebuild: `npm run scrape-partner-skills` is not required for this report; run `npm run build-partner-discrepancies` after re-scraping sources.

Machine-readable twin: `cross-source-diff.json`.
