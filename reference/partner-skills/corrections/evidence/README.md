# Partner skill evidence archive

**Permanent storage for in-game Palpedia screenshots (and related captures).**

## Rules

1. **Keep every screenshot.** Never delete, overwrite, or “replace” a file.
2. **New capture ⇒ new file.** Retakes get a new timestamped name; old files stay.
3. **Commit evidence with the correction** so future sessions still have the binary.
4. Record the path on the pal’s row in `../checklist.json` → `evidence[]`.

## Naming

```
{deckNo}-{pal-slug}--{skill-slug}--{YYYYMMDD-HHmmss}[--n].{ext}
```

Examples:

- `001-lamball--fluffy-shield--20260712-153045.png`
- `071B-vanwyrm-cryst--aerial-marauder--20260713-091200.png`

If deck number is unknown: `000-{pal-slug}--...`.

## Agent duty

When the user attaches or pastes an image, save it here **first**, then update checklist + corrections.
